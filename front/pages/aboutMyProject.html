Sobre Mi Proyecto

Hola, mi nombre es Adriana y soy estudiante de HENRY en la carrera de Desarrollo Web Full Stack, me encuentro actualmente cursando el módulo 2 y este es mi proyecto.

Es una aplicación Full Stack, simulando un cinema, usando los lenguajes HTML, CSS (bajo el preprocesador SASS en su extension "scss"), y JavaScript, este último ejecutandose con Node.js. Utilizando frameworks como Express, Jest para testing; librerías como bootstrap, webpack, axios y mongoose, y módulos de Node.js como morgan y corse. Para la persistencia de datos se ha utilizado MongoDB como base de datos no relacional.

Espero el diseño y funcionalidad de mi aplicación cumplan tus expectativas y disfrutes de lo que estás viendo.
